# AI-Powered Energy Consumption Forecasting

## Overview
This project predicts future energy consumption using machine learning.

## Problem
Energy demand is unpredictable, causing wastage and inefficiency.

## Solution
AI model forecasts future energy usage using historical data.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Time-series forecasting
- MLP model
- Visualization
- Flask API

## Dataset
Time-series energy consumption dataset.

## How to Run
```bash
python main.py
